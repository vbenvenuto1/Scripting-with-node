JavaScript Functions: Intermediate

functions documented

variableTypeOf Pick any character you desire and the return will let you know if its a number or string.

elementIndexOf - Pick any of these words and it will come true.(jello, water, color, football, basketball, protein). Anything outside of this will be false

addSpliceValue - Lets add on to the sentence that is showing. In the first text box type a number in the position you want, second text box type which part you want erased, and lastly the third box go ahead and add a number or words

spliceTheWord - Complete the sentence to your liking. You can remove one of the following words(How, are, you, doing, today).

randomRange - Lets set a min and max values. It will randomly pick a value in between you input.

contacts - Lets play a game where you choose which celebrity you like to contact. Your choices consist of (Johnny Depp, Dwyane Johnson, and Tyra Banks)

headsOrTails - Let's flip a coin! You choose between heads or tails
